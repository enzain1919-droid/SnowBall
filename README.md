# SnowBall

장기투자 포트폴리오를 기록하고, 배당/자가배당/3ETF 은퇴 분배 시나리오를 시뮬레이션하는 Android 앱입니다.
Kotlin과 Jetpack Compose로 개발되었습니다.

## 주요 기능

### 1. 포트폴리오 관리

- 계좌별 자산 관리
- 보유 종목 수동 추가 및 현금 자산 추가
- 종목별 수량, 평균단가, 현재가, 환율 입력
- 매수/매도 거래 내역 관리
- 총 평가금액, 원금, 수익률, 환차익 확인
- 종목별/계좌별 비중 확인

### 2. 홈 대시보드

- 총 자산 요약
- 보유 자산 목록
- 기간별 수익 확인
- 포트폴리오 분석 바로가기
- 가격 데이터 갱신

### 3. 포트폴리오 분석

- 수익 분석
- 해외주식 양도세 분석
- 리밸런싱 점검
- 비중/집중도 점검
- 추세 분석
- 위기 구간 스트레스 지표

### 4. 시뮬레이터

#### 백테스트

- 시작년도/종료년도/시작금 설정
- 리밸런싱 주기 설정
- 적립식 투자 설정
- 배당금 재투입 여부 설정
- 환율 반영 여부 설정
- 자산별 비중 기반 포트폴리오 백테스트
- 누적 금액 변동 그래프
- 손실 그래프
- 연도별 수익률
- 상세 보고서
- 백테스트 저장/불러오기/삭제

#### 배당 시뮬레이션

- 배당 ETF 선택
- 목표 월 배당금/연 배당금/투자금 기준 계산
- 필요 수량 및 필요 투자금 계산
- 예상 월/연 배당금 계산
- 인원별 세금 설정
- 종합소득세 및 건강보험료 추정 반영
- 가격 차트
- 배당 성장 차트
- 1년부터 20년까지 월 평균 배당금 및 총 자산 테이블
- 배당 시뮬레이션 저장/불러오기/삭제

#### 자가배당

- 보유 자산을 매도해 세후 인출 목표액을 맞추는 시뮬레이션
- 연 인출액 및 인출 증가율 설정
- 해외직투 양도세 반영/미반영 선택
- 실현차익에만 해외주식 양도세 적용
- 연간 기본공제 250만원 반영
- 세후 인출 목표액을 맞추기 위한 매도금액 역산
- 연도별 양도세, 매도액, 세후 자산 확인
- 자가배당 그래프
- 자가배당 저장/불러오기/삭제

#### 3ETF 분배

- SCHD, JEPQ, QLD 기반 은퇴 분배 시뮬레이션
- 상장 후 최신일까지의 전체 수정주가 CAGR과 최근 12개월 배당률 자동 계산
- 종목별 최신 수정주가, 배당성장률, 실제 계산기간 자동 표시
- 총 은퇴 자산 및 월 생활비 설정
- 기준 환율 및 초기 주가 설정
- SCHD/JEPQ 배당률·배당성장률과 세 ETF 주가성장률 자동 반영
- QLD 주가성장률 설정
- 현금 이자율, 물가상승률, 세금/건보료 마찰 요율 설정
- 하락장 스트레스 테스트
- 자산 배분 비율 입력 후 `배분 비율 반영` 버튼으로 결과 반영
- 비율 합계가 100%가 아니면 알림 표시
- 20년 뒤 최종 자산 확인
- 초기 확보 주수 확인
- 연차별 누적 자산 막대 그래프
- 그래프 선택 시 자산별 비율과 금액 표시
- 세전배당, 세후배당, 생활비를 포함한 20개년 연도별 은퇴 명세
- 3ETF 분배 저장/불러오기/삭제

#### 시나리오 비교

- 저장된 배당, 자가배당, 3ETF 분배 시나리오를 최대 4개까지 비교
- 연차별 월 현금흐름과 총 자산 비교
- 1년/10년/20년 월 현금흐름, 최종 자산, 누적 현금흐름 요약
- 실제 자산 유출액을 반영한 역산 연수익률 계산
- 종목별 상장 후 전체 수정주가 CAGR을 비중 가중한 과거 연환산 수익률 계산
- 종목 비중과 공통기간 월말 수정주가를 반영한 연환산 변동성 및 최대 낙폭 계산
- 위험 지표의 사용 종목, 공통 분석 기간, 관측 개월 수 표시

### 5. 장기투자 리포트

- 현재 자산과 목표 경로 비교
- 장기투자 계획 설정
- 목표 자산, 연 목표 상승률, 투자 기간 관리
- 목표 경로 이탈 여부 확인
- 월간 장기투자 리포트
- 목표 달성률 및 위기 예상 손실 확인

### 6. 설정 및 백업

- 화면 모드 설정
- 원화/달러 표시 설정
- API 제공자 설정
- KIS/Kiwoom API 키 관리
- 사용자 추가 종목 관리
- 앱 데이터 백업/복원
- 계좌, 거래내역, 목표 설정, 저장한 시뮬레이션, 다운로드한 가격/배당 데이터 백업

- Kotlin

- Material 3
- Gradle Kotlin DSL

- SharedPreferences 기반 로컬 저장

## 빌드 방법

Android Studio에서 프로젝트를 열거나, 터미널에서 다음 명령을 실행합니다.

```bash
./gradlew :app:assembleDebug
```

Windows PowerShell에서는 다음 명령을 사용할 수 있습니다.

```powershell
.\gradlew.bat :app:assembleDebug
```

## 개발 메모

- `local.properties`는 로컬 Android SDK 경로와 민감 정보가 포함될 수 있어 Git에 포함하지 않습니다.
- `.gradle`, `.kotlin`, `build` 등 빌드 산출물은 Git에 포함하지 않습니다.
- API 키는 앱 설정 또는 로컬 속성으로 관리하며 저장소에 직접 커밋하지 않습니다.
